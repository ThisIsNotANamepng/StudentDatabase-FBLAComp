How do I

Set up a scanner

Plug in the barcode scanner into a USB port on the computer. Go to the scanner page. The scanner will automatically scan the barcode and submit it if you click inside the input bar.

Scan an ID

Go to the scanner page and point the scanner at the bar code on the student ID while pulling the trigger

Find the active event

Go to the events page. The current active event is the one shown be default in the dropdown menu.

Change an active event

Go to the events page and choose a new active event frmom the dropdown menu.

Create events

Go to the events page, input a name and select an event type and click submit.

Generate random winners

Go to the winners page and click the "GENERATE RANDOM WINNERS" button.

Find a student

Go to the winners page, enter the students name in the search bar, and click submit.

Analyze reports

Go to the report page.

Manage accounts

Go to the admin page, and use the controls to change passwords, delete accounts, and create accounts.

Reset student points

Go to the admin page, and click the "RESET ALL STUDENT POINTS" button.

Troubleshoot problems

Go to the admin page, and scroll all the way to the bottom to find the log.

Explainers

What is an active event?

An active event is the event in which students scanned by the scanner are entered into.

What are student points?

Students are automatically assigned one point when they attend an event.

Account types

There are three account types.

Admin accounts

Admin accounts can manage accounts and troubleshoot the website, as well as access all parts of the website. They should generally be restricted to only those who need an admin account (i.e. the IT department).

Viewer accounts

Viewer accounts are only allowed to view reports on student involvement and manage events(create events, change the active event and delete events). They should generally be given to school staff who need to guage student involvement.

Scanner accounts

Scanner accounts are only allowed to enter student IDs to the current active event. They should be restricted to computers which scan student IDs at school events.
